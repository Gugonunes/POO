# POO
Projeto Java para programação orientada a objetos.
O projeto é baseado em uma corrida de carros, feita em java para aprender programação orientada a objetos.
Foi feito para a disciplina de Programação Orientada a Objetos, do quarto semestre de computação, 
em dupla com o aluno Pedro Trentin.

O projeto ainda precisa de alguns ajustes finais e algumas validações de entrada.
